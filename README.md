# atulocher  
# 简介：  
atulocher，读作/ætʌlɔ:'kər/，为auto-launcher重组的单词。意为“自动建立者”  
这是一个基于“知识球”+神经网络的人工智能引擎  
设计目的是对抗某些学校的"原创题"  
由于“知识球”的存在，这个人工智能拥有“学会逻辑推理”的能力  
具体 知识球 是什么，见wiki  
***********************************************
源码已经写进include/atulocher下的hpp文件，直接include即可  
## 目前可正常工作的组件：  
* 八叉树  
* 神经网络  
* 中文转向量  
* 逻辑推理（其实就是中文转向量）
